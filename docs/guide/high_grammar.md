

# 补充语法
这里主要介绍vuepress框架提供的一些高级语法，这些语法不是md的标准与法，而是vuepress框架实现的。随着对该框架源码的深入理解以及深度定制，我们也可以自定义更多的高级语法。

## 提示类容器
```
::: tip 提示标题
这是一个提示
:::

::: warning 警告标题
这是一个警告
:::

::: danger 危险警告标题
这是一个危险警告
:::

::: details 详情快标题
这是一个详情块，在 IE / Edge 中不生效
:::
```
::: tip 提示标题
这是一个提示
:::

::: warning 警告标题
这是一个警告
:::

::: danger 危险警告标题
这是一个危险警告
:::

::: details 详情块标题
这是一个详情块，在 IE / Edge 中不生效
:::

## html
vuepress允许你直接写html标签来补充md的不足。
```html
<span style='color:#ce03d0'>我想显示一段有颜色的文字</span>
```
效果：  
<span style='color:#ce03d0'>我想显示一段有颜色的文字</span>

## vue插值表达式
vue插值表达式同样在这里会接受编译
```
{{1 + 1}}
```
效果：  
{{1 + 1}}
